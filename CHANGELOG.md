# Changelog

## [0.1.0] - 2026-04-11

### Added
- Initial release
- Real-time title generation via PreToolUse hook
- Stop hook fallback for sessions without live titles
- Multilingual support (auto-detect or manual language setting)
- Smart throttling (configurable interval and message count)
- Timestamp preservation for transcript files
- Cross-platform support (Linux + macOS)
- `/claude-live-title:config` slash command for configuration
- Claude HUD integration documentation
